# 用户使用手册

## 这是什么工具

这是一个用于生成拼音田字格练习纸的网页工具。你可以输入中文内容，自动生成拼音、田字格练习页和答案页，并支持打印或导出 PDF。

拼音转换库和 PDF 导出相关库都已经放在本地，断网时也可以生成拼音并导出 PDF。完整拼音词典也放在本地，但默认不加载；需要时可以手动打开。

## 基本使用

1. 打开 `index.html`。
2. 在右侧输入练习内容。
3. 在左侧预览 A4 练习纸。
4. 调整颜色、格子大小、每行格数、拼音格式等设置。
5. 点击“打印”或“导出PDF”。

## 两种练习类型

页面右侧有两列内容：

- 看拼音写汉字：题目页显示拼音，隐藏被选中的汉字。
- 看汉字写拼音：题目页显示汉字，隐藏被选中的拼音。

每列下面都有“汉字对应拼音小格区”，用来编辑拼音和选择哪些字作为题目。

## 修改单个字的拼音

在小格区中：

- 上方是汉字。
- 下方是该汉字对应的拼音。

直接修改某个小格下方的拼音即可。修改后预览会自动更新。

## AI 正音

如果希望用 AI 判断多音字和轻声，可以开启“AI 正音”。

需要填写：

- API URL：OpenAI 兼容接口地址，例如 `https://aihubmix.com/v1`。
- Token：接口密钥。
- Model：模型名称，例如 `gpt-4.1-mini`。

开启后，编辑文字时工具会请求 AI 返回每个汉字的数字声调拼音，并更新小格区。AI 请求失败时，会保留本地拼音结果。

注意：这是纯前端本地工具，Token 会保存在当前浏览器的本地存储中，并由浏览器直接发送到你填写的 API URL。请不要在公开网页或共享电脑上填写私人 Token。

## 完整拼音词典

默认情况下，工具只加载 `pinyin-pro` 自带词典，启动更快、占用更小。

如果不使用 AI 正音，但希望本地拼音识别更准确，可以打开“完整拼音词典”开关。打开或关闭该开关后页面会自动刷新一次，让词典状态生效。

## 选择要隐藏的字

点击小格中的汉字：

- 点一次：选中，高亮显示，并在题目页中隐藏对应内容。
- 再点一次：取消选中。

不需要再在文字里输入 `[[` 或 `]]` 标记。文字框会保持干净。

## 空格和分割线

如果文字中有空格，小格区会显示一条高亮分割线，方便快速找到原文中的停顿位置。

如果连续输入多个空格，小格区也只显示一条分割线。

## 放大小格区

小格区默认比较紧凑。

- 点击小格区的空白位置：放大这个区域，能看到更多字。
- 再点一次空白位置：取消放大。
- 点击区域外：也会取消放大。

注意：放大的是容器，不会放大小格里的文字。

## 页面管理

顶部按钮支持：

- 上一页 / 下一页：切换当前编辑页。
- 新建空白页：在当前页后新增一页。
- 删除当前页：删除正在编辑的页。
- 重置为空白页：清空并回到一个空白页。

如果开启“空白页填满田字格”，空白页会显示整页田字格，适合直接打印练字纸。

## 导入和导出文字

点击“导出文字”会保存一个 JSON 文件，包含所有页面内容、手动修改过的拼音和题目选择状态。

点击“导入文字”可以重新载入之前导出的 JSON 文件。

## 打印和导出 PDF

- 打印：调用浏览器打印功能，可以选择打印机或另存为 PDF。
- 导出PDF：直接生成 PDF 文件。

如果 PDF 导出失败，通常是浏览器没有成功加载 PDF 相关库。可以刷新页面，或改用“打印”里的“另存为 PDF”。

## 常见问题

### 修改文字后，手动改过的拼音会丢吗？

通常不会。工具会尽量按汉字顺序保留原来手动修改过的拼音，新加的字才会自动补拼音。

### 拼音不准怎么办？

在小格区直接修改对应字下面的拼音。

### 可以显示或隐藏标点吗？

可以。使用“显示标点”开关控制预览页是否显示标点。

### 可以不要答案页吗？

可以。关闭“生成答案页”即可。

### 拼音想改成数字声调怎么办？

在“拼音格式”中选择“数字声调”。也可以选择“声调符号”或“无声调”。
